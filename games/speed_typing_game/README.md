# Speed Typing Game

A fast typing challenge built with HTML, CSS, and JavaScript using the SlidePlay dark theme.

## Features

- Three difficulty levels: **Easy**, **Medium**, and **Hard**
- Hard mode requires fast typing under a shorter time limit
- Real-time scoring and typing progress
- Modern dark UI with magenta and cyan accent colors
- Keyboard input to test your speed and accuracy

## How to Play

1. Open `speed_typing_game.html` in your browser
2. Choose a difficulty level
3. Click **Start** to begin the timer
4. Type the displayed phrase exactly into the input field
5. Press **Reset** to restart anytime

## Difficulty

- **Easy**: More time and shorter phrases
- **Medium**: Moderate time with longer sentences
- **Hard**: Shortest time and fastest phrase pacing

## Files

- `speed_typing_game.html` - Game layout and controls
- `style.css` - Theme and visual styling
- `script.js` - Typing game logic and timer behavior

## Customization

To update the typing prompts, edit the `challenges` array in `script.js`.

Enjoy practicing your typing speed! ⌨️
