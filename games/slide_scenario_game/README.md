Slide Scenario Game

What it does:
- Turns slide content into scenario prompts.
- Learners choose the best next action for each slide-based situation.
- Immediate feedback shows the correct choice in green and the wrong choice in red.
- Scores reward accuracy, speed, and streaks.

How slide content is used:
1) Try uploaded slide text first.
2) Fall back to generated quiz content if present.
3) Use built-in scenario examples if no slide content exists.

Main files:
- index.html: UI structure.
- style.css: polished scenario-based layout.
- script.js: modular scenario extraction, choice handling, validation, and scoring.

Extension ideas:
- Add branching paths based on answer history.
- Add timer pressure and power-ups.
- Add multiplayer challenge mode.
